Task based communication built for professional teams


This functionality describes about the flow how user can communicate with other users. Whhen user has successfully logged in and is ready to communicate. User opens web application. User goes to the Chat option. User selects the subject of task. User adds people with whom he wants to communicate. User sends messages through the message box or clicks on audio call option if he/she wants audio call.
